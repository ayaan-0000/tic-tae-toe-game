## 📖 Project Explanation

The game renders a 3x3 board with aligned grid formatting using `iomanip`, and alternates turns between two players, tracking move coordinates and determining the winner once the board is complete.

## 🧩 Things Used

- **`iomanip`** — aligned grid formatting for the board display
- **2D char array** — represents the 3x3 game board

## 🚀 Build & Run

```bash
g++ "tic tac toe game.cpp" -o tictactoe
./tictactoe
```

## ⭐ Honest Review

**Strengths:** A clean, working implementation of a classic problem with genuinely nice formatted output — the board rendering is more polished than most beginner tic-tac-toe implementations.

**Areas for improvement:** As a single small game, this is best presented as part of a broader "mini C++ projects" collection rather than as its own repository — bundling it with similar small exercises under one well-organized repo would give it more context than it has standing alone.
